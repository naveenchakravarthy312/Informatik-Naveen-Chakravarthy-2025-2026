# Übung U4_3 – Farben, Rahmen und Größen

**Kurzbeschreibung**
Gestalte einen Schriftzug nur mit HTML und CSS.

**Aufgabe**
- Erstelle ein HTML‑Grundgerüst.
- Gestalte einen Schriftzug mit einem einzigen `<h1>`.
- Nutze nur CSS (style‑Attribut) für Farben, Rahmen und Größen.
- Keine Bilder verwenden.

**Abgabe**
- ZIP‑Datei mit der HTML‑Datei.

**Was du lernst**
- Farben und Rahmen in CSS
- Größenangaben anwenden
- Gestaltung ohne Bilder
