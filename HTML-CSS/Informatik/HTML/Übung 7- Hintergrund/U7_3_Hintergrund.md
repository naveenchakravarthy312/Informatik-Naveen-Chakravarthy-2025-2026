# Übung U7_3 – Hintergrund

**Kurzbeschreibung**
Gestalte den Seitenhintergrund und bearbeite zwei vorgegebene Bereiche.

**Aufgabe**
- Öffne die vorgegebene HTML‑Datei (z. B. carnary_wharf.html).
- Bette ein Hintergrundbild ein.
- Positioniere das Bild passend.
- Passe die Gestaltung der beiden Bereiche an.

**Abgabe**
- ZIP‑Datei mit HTML‑Datei und Bilddateien.

**Was du lernst**
- Hintergrundbilder in CSS
- Positionierung von Hintergründen
- Gestaltung von Bereichen
