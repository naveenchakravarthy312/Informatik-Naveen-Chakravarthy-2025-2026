Google Font

So macht man Google Fonts in HTML:

  1. Eine Font auswählen von Google Fonts
  2. Bekomme den Link zum Font. So kann es aussehen:

    • link rel="preconnect" href="https://fonts.googleapis.com">
      link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
      link href="https://fonts.googleapis.com/css2?family=Bitcount+Prop+Double+Ink:wght@100..900&display=swap" rel="stylesheet">

  3. Font in HTML einbinden. Das kann irgenwo in der HTML-Datei stehen. (z.B. in einem style Tag Block)
