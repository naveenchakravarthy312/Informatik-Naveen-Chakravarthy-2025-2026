## Projekt: „Meine lokale Entdecker‑Seite“

### Ziel
Du erstellst eine eigene kleine Website über einen Ort in deiner Umgebung (z. B. Stadtteil, Park, Verein, Museum, Café, Denkmal). Du zeigst, dass du HTML‑Struktur und CSS‑Gestaltung sicher anwenden kannst.

### Umfang
- mindestens 3 Seiten (z. B. `index.html`, `info.html`, `kontakt.html`)
- einheitliche Navigation auf allen Seiten
- klares Layout und gute Lesbarkeit

### Mindestanforderungen (Pflicht)
- Die folgenden Anforderungen müssen ohne KI erstellt werden.
- semantische Struktur: `header`, `nav`, `main`, `section`, `footer`
- mindestens 2 Bilder mit Quellenangabe/Lizenz
- eine Liste (z. B. Öffnungszeiten, Highlights)
- eine Tabelle (z. B. Fakten, Preise, Zeiten)
- CSS extern in einer eigenen Datei
- saubere Ordnerstruktur:
  - `css/` (Styles)
  - `img/` (Bilder)

### Gestaltung
- einheitliche Farben und Schriftstile
- sinnvolle Abstände
- lesbare Schriftgrößen
- keine überladenen Seiten

### Abgabe
- kompletter Projektordner
- Startseite heißt `index.html`
- Quellen/Lizenzen in `quellen.html` oder im Footer
- Abgabe bis Fr, 27.03.2026
- Kurzvorstellung in der Stunde davor (2–3 Minuten)
- Erweiterungen sind erlaubt (z. B. zusätzliche Seiten), dürfen aber die Mindestanforderungen nicht ersetzen.

### Meilensteine (empfohlen)
- Konzept/Struktur bis Mo, 16.02.2026
- Rohfassung mit 3 Seiten bis Mo, 09.03.2026
- Finalisierung bis Do, 26.03.2026

### Abgabeformat
- Projektordner als ZIP abgeben
